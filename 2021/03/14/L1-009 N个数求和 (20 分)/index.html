L1-009 N个数求和 (20 分)

题目链接

解题思路：

水题，模拟就行，主要是求最大公因数，用__gcd就可以，这里有个坑，在代码里标明了，如果是 long long int sum=a*lcm/b+x*lcm/y; 会wa掉一组样例（3分）

我猜是因为lcm和a都太大，如果先乘再除的话会爆long long？所以先除再乘就可以了，不过b和y也都是长整型的，为啥 long long int lcm=b*y/flag; 不会爆？（如果爆了我也可以理解，但是没有）。不过确实lcm要比b*y要大不少。
